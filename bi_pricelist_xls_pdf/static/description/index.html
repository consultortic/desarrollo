Export Customer Pricelist in Excel/PDF odoo app helps users to export multiple customer's pricelist (Group By Partner) into multiple file types like PDF or Excel.

Features

Export Multiple Customers Pricelist.

User can export pricelist for multiple selected customer at once.

Export in PDF or XLS File Type.

User can export pricelist in multiple file type like PDF or XLS.
Demo Request Video Tutorial

Export Customers Pricelist

Under customer list user can select multiple customer, Under action menu user can see "Bi Customers Pricelist" menu to export selected customers pricelist.

User have option to export file in PDF or XLS file type.

Customers Pricelist PDF Report

On clicking "PDF REPORT" button user can see generated PDF report for selected customers group by partner.

Customers Pricelist XLS Report

On clicking "EXCEL REPORT" button user can see generated XLS report for selected customers group by partner and sheet.
